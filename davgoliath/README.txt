Building GoliathRequirements	- CodeWarrior Pro 7.1 is required to build GoliathHow to Build Goliath1).  Get OpenSSL 0.95; make sure it's in the same folder as the goliath source.       Build the MacOS targets.2).  Checkout the DAVLib module from SourceForge; check it out in the same folder      as the goliath source.  3).  Decode the resource files; in the Resources sub directory decompress the .hqx files4).  Create the Goliath CodeWarrior project file - import the Goliath.mcp.xml file.  Name it    'Goliath.mcp'
